===================================================================================================
Avant d'éxécuter les programmes python assurez vous d'avoir
installé les bibliothèques nécéssaires. (assurez vous que python et pip sont installés sur votre machine)

- lancez le fichier "install packages.bat"

Si après l'éxécution des programmes, des erreurs d'importations sont toujours présentes,
installez manuellement les bibliothèques contenues dans le fichier "requirements.txt"
